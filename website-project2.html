Website Project 2 — Split Layout with Sticky Sidebar

A modern, app-like web experience featuring a persistent sidebar and a clean content canvas. Built for clarity, speed, and delightful micro-interactions—fully responsive and image-free.

Overview

Website Project 2 explores a split layout with a sticky navigation sidebar. The design feels like a lightweight app: quick to scan, effortless to navigate, and highly adaptable to content-heavy portfolios or documentation.

Split Layout Sticky Nav Responsive Accessible No Images
View Mockup Back to Portfolio

Key Features

  • Sticky left sidebar for fast navigation across sections.
  • Generous spacing, crisp typography, and soft gradients.
  • Component-driven: buttons, cards, panels, and grids.
  • Pure HTML/CSS (no external images) for speed and maintainability.
